# de.wenzlaff.dump1090

Java Wrapper für die Dump1090 Schnittstelle.

Dieses Projekt ist eine Javaschnittstelle für den Zugriff auf die Daten von Dump1090 per Java Objekte.

Folgende Klassen gibt es:

Flugzeug
Flugzeuge
Luftnotfall
